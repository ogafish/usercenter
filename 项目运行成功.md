## 📋 可用的 API 接口

- `POST /api/user/register` - 用户注册
- `POST /api/user/login` - 用户登录
- `POST /api/user/logout` - 用户注销
- `GET /api/user/current` - 获取当前用户
- `GET /api/user/search` - 搜索用户（需要管理员权限）
- `POST /api/user/delete` - 删除用户（需要管理员权限）

## 🔍 故障排查

### 如果后端无法启动
1. 检查 MySQL 服务是否运行
2. 检查端口 8080 是否被占用
3. 检查 JAVA_HOME 是否设置为 JDK 1.8

### 如果前端无法启动
1. 检查端口 8000 是否被占用
2. 检查 Node.js 版本
3. 重新安装依赖: `npm install`

### 如果登录失败
1. 检查数据库连接
2. 确认用户名密码正确
3. 查看后端日志
