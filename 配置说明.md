# 用户中心项目配置指南

## 一、环境准备

### 必需软件
1. **JDK 1.8** - Java开发环境
2. **Maven 3.6+** - Java项目构建工具
3. **MySQL 5.7+** - 数据库
4. **Node.js 14+** - 前端运行环境
5. **IDE** - 推荐 IntelliJ IDEA 或 VSCode

---

## 二、后端配置步骤

### 1. 数据库配置

#### 创建数据库
```bash
# 登录MySQL
mysql -u root -p

# 执行SQL文件创建数据库和表
source user-center-backend-master/user-center-backend-master/sql/create_table.sql
```

或者手动执行：
- 创建数据库 `yupi`
- 创建用户表 `user`
- 插入示例数据

#### 修改数据库连接信息
编辑文件：`user-center-backend-master/user-center-backend-master/src/main/resources/application.yml`

```yaml
datasource:
  driver-class-name: com.mysql.cj.jdbc.Driver
  url: jdbc:mysql://localhost:3306/yupi?useSSL=false&serverTimezone=Asia/Shanghai&characterEncoding=utf8
  username: root
  password: 你的数据库密码  # 修改为你的MySQL密码
```

### 2. 启动后端项目

#### 方式一：使用IDE
1. 用 IntelliJ IDEA 打开 `user-center-backend-master/user-center-backend-master` 目录
2. 等待 Maven 依赖下载完成
3. 找到 `UserCenterApplication.java` 主类
4. 右键运行 Run 'UserCenterApplication'

#### 方式二：使用命令行
```bash
cd user-center-backend-master/user-center-backend-master

# Windows
mvnw.cmd spring-boot:run

# Linux/Mac
./mvnw spring-boot:run
```

### 3. 验证后端启动
访问：http://localhost:8080/api
如果看到页面或返回信息，说明后端启动成功。

## 三、前端配置步骤

### 1. 安装依赖
```bash
cd user-center-frontend-master/user-center-frontend-master

# 使用 npm
npm install

# 或使用 yarn（推荐）
yarn install
```

### 2. 启动前端项目
```bash
# 开发环境启动
npm run start
# 或
yarn start
```

### 3. 访问前端
浏览器自动打开：http://localhost:8000

## 四、常见问题

### 1. 后端启动失败
- 检查 MySQL 是否启动
- 检查数据库名称、用户名、密码是否正确
- 检查端口 8080 是否被占用
- 检查 JDK 版本是否为 1.8

### 2. 前端启动失败
- 删除 node_modules 文件夹，重新安装依赖
- 检查 Node.js 版本是否符合要求
- 检查端口 8000 是否被占用

### 3. 前后端联调失败
- 确认后端已启动在 8080 端口
- 检查前端代理配置
- 检查浏览器控制台是否有跨域错误

### 4. 数据库连接失败
- 确认 MySQL 服务已启动
- 检查数据库连接字符串是否正确
- 检查防火墙是否阻止了 3306 端口

## 五、项目结构

### 后端结构
```
src/main/java/com/yupi/usercenter/
├── common/          # 通用类（返回结果、错误码等）
├── constant/        # 常量
├── controller/      # 控制器层
├── exception/       # 异常处理
├── mapper/          # 数据访问层
├── model/           # 实体类
├── service/         # 业务逻辑层
└── UserCenterApplication.java  # 启动类
```

### 前端结构
```
src/
├── components/      # 组件
├── pages/          # 页面
├── services/       # API 服务
└── app.tsx         # 应用入口
```
