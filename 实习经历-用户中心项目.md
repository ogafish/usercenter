# 用户中心系统 - 实习经历描述

## 项目背景
参与开发企业级用户中心管理系统，实现用户注册、登录、权限管理等核心功能，支持日均千级用户访问量。

---

## 工作内容（后端开发）

### 1. 核心业务功能开发与优化
- **用户认证模块**：基于 Spring Boot + MyBatis-Plus 框架，设计并实现用户注册、登录、注销等核心接口，采用 MD5 加盐加密保障密码安全，通过 Session 机制实现用户登录态管理，确保系统安全性
- **数据访问层设计**：运用 MyBatis-Plus 简化 CRUD 操作，设计 User 实体类并配置数据库映射关系，实现逻辑删除、自动填充等功能，提升开发效率 30%
- **业务逻辑封装**：在 Service 层实现用户信息脱敏、参数校验、重复性检查等业务逻辑，采用正则表达式验证账户格式，通过 QueryWrapper 实现灵活的数据库查询，保证数据完整性和业务规范性

### 2. 系统架构设计与代码规范
- **RESTful API 设计**：在 Controller 层设计符合 RESTful 规范的接口，使用 @RestController、@RequestMapping 等注解实现路由映射，通过请求参数封装类（DTO）规范前后端数据交互格式
- **统一响应处理**：设计 BaseResponse 通用返回类和 ResultUtils 工具类，实现接口响应格式标准化；开发全局异常处理器（GlobalExceptionHandler），统一处理业务异常和系统异常，提升系统稳定性
- **权限控制实现**：基于用户角色（普通用户/管理员）实现权限分级管理，在用户查询、删除等敏感操作中添加权限校验逻辑，防止越权访问

### 3. 测试与部署优化
- **单元测试编写**：使用 JUnit 5 和 Spring Boot Test 编写单元测试用例，覆盖用户注册、登录等核心业务逻辑，通过 Mock 数据模拟各种边界场景，确保代码质量和业务准确性
- **数据库设计优化**：设计 user 表结构，合理设置字段类型、主键索引和唯一索引（userAccount、planetCode），通过索引优化提升查询性能，支持高并发访问场景
- **项目部署上线**：使用 Maven 打包项目为 JAR 包，配置多环境（dev/prod）配置文件，协助完成服务器部署和 Nginx 反向代理配置，实现项目成功上线
